T-Rex Runner
============

Arcade Hub loads the upstream runner from:

- vendor/wayou-t-rex-runner/index.js
- vendor/wayou-t-rex-runner/assets/default_100_percent/100-offline-sprite.png
- vendor/wayou-t-rex-runner/assets/default_200_percent/200-offline-sprite.png

The upstream code is adapted through the local Arcade Hub adapter in main.js for
page controls, fullscreen behavior and XP bridge events. Provenance and license
details are recorded in ../../third_party/t-rex/.
