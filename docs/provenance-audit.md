# Provenance audit

Audit date: 2026-06-22. This report records provenance evidence already present in the repository and its Git history. It does not independently certify ownership or compare local files with upstream releases.

Confidence means:

- **HIGH** — the repository contains direct, mutually consistent evidence such as a source URL, named author/project and license text, or a continuous first-party development history.
- **MEDIUM** — the repository contains a source or license declaration, but lacks an exact upstream revision, independent provenance, or another part of the chain of custody.
- **LOW** — origin, author or applicable license cannot be established from repository evidence. Such entries remain high risk.

`UNKNOWN` is used whenever the repository does not prove the requested fact. A Git committer is not treated as the author or copyright holder of imported material merely because they committed it.

## Confidence summary

### HIGH confidence components

- Poker code (excluding chip PNGs)
- Catch the Cats
- 2048
- Canvas Tetris
- Poppins remote font dependency
- Postgres.js
- ws

### MEDIUM confidence components

- Pacman and T-Rex Runner: named upstreams and local license claims exist, but the applicable license for the unknown imported revision is not resolved.
- Breakout, Flappy Bird, Minesweeper, Pong and Snake: colocated MIT declarations and import records exist, but provenance is self-declared and no independent upstream/creation evidence exists.
- Asteroids, Space Invaders, Frogger, Galaga, Missile Command, Simon, Connect Four, Whac-A-Mole, Memory Match, Sokoban, Brick Breaker, Tic-Tac-Toe, Hangman, Solitaire and Sudoku: central source/license declarations and creation commits exist, but no independent provenance or colocated license exists.
- Freedoom browser game and data, and the Dwasm/PrBoomX build: projects and licenses are named, but exact releases/build correspondence are missing.
- Synthesized Web Audio effects, Klaro, Supabase JavaScript client and the remaining locked npm set.

### LOW confidence / HIGH RISK components

Provenance confidence and licensing risk are related but not identical. A self-declaration can provide **MEDIUM** provenance evidence while still leaving the component **HIGH RISK** because authority to make that declaration is unproved.

- **MEDIUM confidence / HIGH RISK:** Pacman and T-Rex Runner, because the named upstream is known but the license for the exact imported revision is unresolved.
- **MEDIUM confidence / HIGH RISK:** Breakout, Flappy Bird, Minesweeper, Pong, Snake, Asteroids, Space Invaders, Frogger, Galaga, Missile Command, Simon, Connect Four, Whac-A-Mole, Memory Match, Sokoban, Brick Breaker, Tic-Tac-Toe, Hangman, Solitaire and Sudoku, because only internal source/license claims and addition commits exist.
- **MEDIUM confidence / HIGH RISK:** Dwasm/PrBoomX compiled runtime, because exact corresponding source and build provenance are absent.
- **LOW confidence / HIGH RISK:** Google libarchive/Comlink bundle, because only a partial embedded header exists and exact upstream plus complete JS/WASM license scope are unknown.
- **LOW confidence / HIGH RISK:** Poker chip atlas and derived PNGs, because source, creator and license are unknown.
- **LOW confidence / HIGH RISK:** Game catalogue SVG thumbnails, because addition commits exist but author/source and explicit asset license are unknown.
- **LOW confidence / HIGH RISK:** Portal OpenGraph SVG, because an addition commit exists but author/source and explicit asset license are unknown.

## Coverage notes

- Scanned game directories and pages, `about/licenses.html`, repository documentation, `third_party/`, notices, lockfiles, relevant comments/headers and Git history.
- Scanned tracked SVG, PNG, audio, font, WASM, `.data` and `.bin` files. No tracked audio or font binaries were found.
- Inline SVG UI icons are part of HTML/application source and have no separate provenance declarations; they therefore inherit the provenance status of their containing first-party file unless future evidence identifies a separate source.
- Hosted analytics/advertising scripts are external services, not copied game assets; they are documented in `docs/third-party-notices.md` and are outside this game/asset provenance table.
